# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✓         |

## Reporting a vulnerability

**Do not open a public GitHub issue for a security vulnerability.**

Email: **security@opensyber.cloud** (or fall back to `support@opensyber.cloud`)

Include:
- Description and reproduction steps
- The version affected (`opensyber-mcp-watch --version`)
- Suggested fix if you have one

## SLA

- **48 hours** to first reply
- **7 days** to confirm or dispute
- **30 days** to ship a fix for confirmed criticals/highs
- **90 days** maximum embargo before public disclosure

## Out of scope

- Vulnerabilities in dependencies (file upstream)
- Vulnerabilities in MCP servers you scan (file with the server author)
- Issues requiring physical access to the user's machine

## PGP

Email is fine to start. PGP key available on request.

## Hall of fame

Researchers who report valid vulnerabilities will be credited in release notes unless they ask otherwise.
